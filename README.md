# Tui Grid Movement

The absolute basic app for controlling an x,y point in a 2D plane.

![](https://user-images.githubusercontent.com/1455572/104426796-5e0ca900-55bd-11eb-8fdd-5441bfb5ce9e.png)

### Dev

``` shell
make dev
```

### Package

``` shell
make package
```

# Makefile

``` makefile
dist
node_modules
yarn.lock
```

``` makefile
dev:
    yarn dev
package:
    yarn package
init:
    yarn install
```
